1. Installing Python

  • Go to python.org and download the latest version.
  • Ensure you check "Add Python to PATH" during installation.

2. Starting IDLE

Open IDLE after installation to begin coding. It uses a REPL (Read-Eval-Print Loop).

3. Understanding print() Function

The print() function displays output to the screen.

Syntax:

print(value1, value2, ..., sep=' ', end='\n')
  • Can print text, numbers, variables, or expressions.
  • Text must be in quotes: 'text' or "text".
  • Multiple values are separated with commas.

Examples:

print("Hello World")
print(10 + 5)
print("The sum is", 10 + 5)

4. Using print() in REPL

print("Hello")

11. Understanding input() Function

input() takes user input as a string.

variable = input("Prompt message")
  • To do math, convert input using int() or float().
name = input("Enter your name: ")
print("Hello", name)

num1 = input("Enter a number: ")
num2 = input("Enter another number: ")
print(num1 + num2)  # concatenates strings

# Convert to integers
num1 = int(input("Enter a number: "))
num2 = int(input("Enter another number: "))
print(num1 + num2)

12. String Concatenation Without Conversion

x = input("Enter first number: ")
y = input("Enter second number: ")
print("The sum is:", x + y)  # Output will be string concatenation (e.g. 105)

13. Converting Input to Numbers

x = int(input("Enter first number: "))
y = int(input("Enter second number: "))
print("The sum is:", x + y)  # Correct addition
