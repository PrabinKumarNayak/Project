Topic: Conditional Statements

A. Introduction to Conditional Statements

Conditional statements allow a program to make decisions based on conditions.

1. if Statement

The if statement runs a block only if a condition is true.

if condition:
    # block of code
Example: 
age = 20
if age > 18:
    print("You are eligible")

2. Relational Operators

Used to compare two values and return True or False.

OperatorMeaningExampleResult
>Greater than5 > 3True
<Less than3 < 2False
>=Greater or equal5 >= 5True
<=Less or equal4 <= 5True
==Equal to5 == 5True
!=Not equal5 != 3True

Programs Using Simple if

Program 1

num = int(input("Enter a number: "))
if num > 18:
    print("Your Name")

Program 2

num = int(input("Enter a number: "))
if num > 18:
    print("First Name")
else:
    print("Surname")

Boolean Expressions

print(5 > 3)     # True
print(4 == 7)    # False
print(10 != 5)   # True

Importance of Indentation

age = 20
if age > 18:
    print("Adult")
print("Done")
