Unit 5: Arrays in C

1. Introduction to Arrays

An array is a collection of elements of the same data type stored in contiguous memory locations. It allows storing multiple values under a single variable name.

2. Types of Arrays

  • One-dimensional (1D) array: A list of elements
  • Two-dimensional (2D) array: A matrix of elements (rows and columns)

3. Declaring and Initializing Arrays

int arr[5];                    // declaration
int arr[5] = {10, 20, 30, 40, 50}; // initialization

Indexing starts from 0 to size-1.

4. Accessing Array Elements

Use index to access or modify an element:

arr[0] = 100;    // assigns 100 to the first element
printf("%d", arr[2]); // prints third element

5. Traversing Arrays Using Loops

Array elements are often accessed using loops:

for (int i = 0; i < 5; i++) {
    printf("%d ", arr[i]);
}

6. Two-Dimensional Arrays

int matrix[2][3] = {
    {1, 2, 3},
    {4, 5, 6}
};

Accessing elements in 2D array:

printf("%d", matrix[1][2]); // prints 6

7. Passing Arrays to Functions

To pass an array to a function, pass its name without brackets:

void printArray(int arr[], int size) {
    for (int i = 0; i < size; i++) {
        printf("%d ", arr[i]);
    }
}

8. Limitations of Arrays

  • Fixed size — cannot grow or shrink dynamically
  • All elements must be of the same data type

Summary – Key Exam Concepts

  • Concept of 1D and 2D arrays
  • Syntax for declaration and initialization
  • Accessing and modifying elements using index
  • Looping through arrays with for loop
  • Passing arrays to functions
