Unit 10: Summary of C Programming

This unit provides a quick summary of the concepts covered in the C programming tutorial. It can be used for quick revision before exams or while preparing assignments.

Unit 1: Basic Programs

  • Understanding structure of a C program
  • Header files and main function
  • Basic I/O using printf() and scanf()
  • Use of arithmetic operators

Unit 2: Selection Control

  • Use of if, if-else, nested if
  • switch statement for multi-way decisions
  • Relational and logical operators

Unit 3: Looping Statements

  • for, while, and do-while loops
  • Control statements like break and continue

Unit 4: Functions

  • User-defined vs library functions
  • Function declaration, definition, and call
  • Passing parameters and recursion

Unit 5: Arrays

  • 1D and 2D arrays
  • Array indexing and traversal
  • Passing arrays to functions

Unit 6: Strings

  • String declaration and initialization
  • String functions from <string.h>
  • String input/output methods

Unit 7: Structures

  • Defining and accessing structure members
  • Nested structures and arrays of structures
  • Passing structures to functions

Unit 8: Pointers

  • Pointer declaration and usage
  • Pointer arithmetic and arrays
  • Function arguments and dynamic memory

Unit 9: File Handling

  • Opening, reading, writing and closing files
  • File modes and basic file functions
  • Using FILE pointer and fprintf, fscanf, etc.

Note: For each unit, students are advised to revise key syntax and practice writing programs themselves.
