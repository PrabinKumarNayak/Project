Unit 3: Looping Statements in C

1. Introduction

In programming, loops are used to execute a block of code repeatedly as long as a specified condition is true. C provides three primary loop constructs: for, while, and do-while.

2. Types of Loops

  • Entry-Controlled Loops: Condition is checked before the loop body is executed (e.g., for and while loops).
  • Exit-Controlled Loops: Loop body is executed at least once before condition is checked (e.g., do-while loop).

3. The for Loop

The for loop is best when the number of iterations is known in advance.

for (initialization; condition; increment/decrement) {
    // loop body
}
  • initialization: Sets the starting value.
  • condition: Checked before every iteration. Loop runs if true.
  • increment/decrement: Updates the loop variable after each iteration.

4. The while Loop

Used when the number of iterations is not known in advance.

while (condition) {
    // loop body
}

Note: Condition is checked before entering the loop body. If the condition is false initially, the loop will not run.

5. The do-while Loop

Executes the loop body at least once, and then checks the condition.

do {
    // loop body
} while (condition);

Note: The condition is checked after one execution.

6. Loop Control Statements

  • break: Terminates the loop immediately.
  • continue: Skips the current iteration and continues with the next.

7. Nested Loops

Using one loop inside another is called nesting. Commonly used for matrix operations or pattern printing.

for (int i = 0; i < n; i++) {
    for (int j = 0; j < m; j++) {
        // inner loop body
    }
}
