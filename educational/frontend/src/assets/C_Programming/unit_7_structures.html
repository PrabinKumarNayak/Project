Unit 7: Structures in C

1. Introduction

A structure in C is a user-defined data type that allows grouping variables of different types under one name. It is useful for representing complex data models.

2. Defining a Structure

struct Student {
    int roll;
    char name[20];
    float marks;
};

3. Declaring Structure Variables

struct Student s1, s2;

Or combined with definition:

struct Student {
    int roll;
    char name[20];
    float marks;
} s1, s2;

4. Accessing Structure Members

Use the dot operator:

s1.roll = 101;
printf("%d", s1.roll);

5. Initializing Structures

struct Student s1 = {101, "Ravi", 87.5};

6. Nested Structures

A structure inside another structure:

struct Date {
    int day, month, year;
};

struct Student {
    int roll;
    struct Date dob;
};

7. Array of Structures

You can create multiple structure variables using an array:

struct Student s[50];

Useful for storing data of many students.

8. Passing Structure to Functions

  • By value: A copy is passed, original remains unchanged.
  • By reference: Using pointers to reflect changes on original data.

9. typedef with Structure

Simplifies structure declarations:

typedef struct Student {
    int roll;
    char name[20];
} STU;

STU s1;
