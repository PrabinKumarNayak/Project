Unit 8: Pointers in C

1. Introduction

A pointer is a variable that stores the memory address of another variable. Pointers are powerful tools for memory management, dynamic allocation, and function argument passing.

2. Declaring and Initializing Pointers

int *ptr;      // declaration
int a = 10;
ptr = &a;     // ptr holds the address of variable a

3. Pointer Operators

  • & — Address-of operator (gets address of variable)
  • * — Dereference operator (accesses value at address)

4. Accessing Values Using Pointers

printf("%d", *ptr);   // prints value of a

5. Pointer Arithmetic

Pointers can be incremented/decremented:

ptr++;
ptr--;

Pointer arithmetic depends on the data type (increments by size of type).

6. Pointers and Arrays

int arr[3] = {10, 20, 30};
int *p = arr;   // points to first element
printf("%d", *(p + 1));  // prints second element

7. Pointer to Pointer

A pointer that stores the address of another pointer.

int a = 10;
int *p = &a;
int **q = &p;

8. Pointers and Functions

Used to pass addresses to functions (pass by reference):

void modify(int *x) {
    *x = *x + 1;
}

9. Dynamic Memory Allocation (Intro)

Using malloc(), calloc(), free() from <stdlib.h> to allocate and deallocate memory at runtime.

Summary – Key Exam Concepts

  • Concept of pointers and syntax
  • Pointer declaration and initialization
  • Dereferencing and address-of operators
  • Relation between pointers and arrays
  • Pointer to pointer, and function arguments using pointers
  • Basic pointer arithmetic
