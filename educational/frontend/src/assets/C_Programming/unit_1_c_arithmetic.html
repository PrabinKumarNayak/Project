Unit 1: Basic Programs and Arithmetic Operators

1. Introduction to C Programming

C is a procedural and structured programming language developed by Dennis Ritchie in 1972. It is widely used in system-level programming, embedded systems, and application development.

2. Structure of a C Program

#include <stdio.h>
int main() {
    // Variable Declarations
    // Logic and statements
    return 0;
}
  • #include <stdio.h>: Includes standard input/output functions.
  • main(): The entry point of a C program.
  • return 0;: Indicates successful program execution.

3. Variables and Data Types

C is a statically typed language. Common data types include:

Data Type Description Example
int Integer values int age = 21;
float Decimal values float pi = 3.14;
char Single character char grade = 'A';

4. Input and Output Functions

printf() - Output

Used to display output on the screen.

printf("Sum = %d", sum);

scanf() - Input

Used to take input from the user.

scanf("%d", &a);

Note: Use & to provide the address of the variable.

5. Arithmetic Operators

Operator Description Example
+ Addition a + b
- Subtraction a - b
* Multiplication a * b
/ Division (quotient) a / b
% Modulus (remainder) a % b

6. Integer vs Float Division

  • Integer division: 5 / 2 results in 2
  • Float division: 5.0 / 2 or (float)5 / 2 results in 2.5

7. Constants

Constants are values that do not change during program execution.

#define PI 3.14

8. Operator Precedence

  1. *, /, % (Higher precedence)
  2. +, - (Lower precedence)

Operators with the same precedence are evaluated from left to right.

9. Modulus and Digit Extraction

  • n % 10: Gives the last digit
  • n / 10: Removes the last digit

This is useful for operations like finding the last digit, summing digits, or reversing numbers.
