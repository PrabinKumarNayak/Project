Unit 2: Selection Control Statements

1. Introduction

In C programming, selection control statements are used to make decisions based on conditions. They allow the program to choose different paths of execution depending on whether a condition is true or false.

2. Relational Operators

Relational operators are used to compare two values:

Operator Meaning Example
== Equal to a == b
!= Not equal to a != b
> Greater than a > b
< Less than a < b
>= Greater than or equal to a >= b
<= Less than or equal to a <= b

3. Logical Operators

Logical operators are used to combine multiple conditions:

Operator Meaning Example
&& Logical AND (a > 0 && b > 0)
|| Logical OR (a > 0 || b > 0)
! Logical NOT !(a == b)

4. if Statement

The if statement executes a block of code only if the specified condition is true.

if (condition) {
    // statements
}

5. if-else Statement

Provides two paths: one if condition is true, another if it is false.

if (condition) {
    // true block
} else {
    // false block
}

6. Nested if-else

Nested decision-making inside an if or else block.

if (condition1) {
    if (condition2) {
        // inner block
    } else {
        // inner else block
    }
} else {
    // outer else block
}

7. else-if Ladder

For multiple conditions checked sequentially.

if (condition1) {
    // block1
} else if (condition2) {
    // block2
} else if (condition3) {
    // block3
} else {
    // default block
}

8. switch Statement

Efficient for comparing a single variable against multiple constant values.

switch (expression) {
    case value1:
        // statements
        break;
    case value2:
        // statements
        break;
    default:
        // default statements
}
  • break exits the switch block after a match.
  • default executes when no match is found.
