Unit 6: Strings in C

1. Introduction

A string in C is a sequence of characters terminated by a null character ('\0'). Strings are stored in character arrays.

2. Declaring Strings

char str[20];             // uninitialized
char str[] = "Hello";    // initialized with string literal

Each character is stored in a separate array element, ending with '\0'.

3. Reading Strings

Using scanf (single word):

scanf("%s", str);

Using gets (entire line): (Note: gets() is unsafe and deprecated in modern C.)

gets(str);

Using fgets (safer):

fgets(str, sizeof(str), stdin);

4. Writing Strings

Using printf:

printf("%s", str);

5. Common String Handling Functions (in <string.h>)

Function Description
strlen(str) Returns the length of the string
strcpy(dest, src) Copies one string to another
strcat(dest, src) Appends one string to another
strcmp(str1, str2) Compares two strings

6. String vs Character

  • Character: Single character like 'A'
  • String: Collection of characters like "ABC"

7. String as Array

Each character including '\0' is stored in an array index:

char str[] = "Hi";  // stored as {'H', 'i', '\0'}

8. Modifying Strings

We can access and modify individual characters using indexing:

str[0] = 'h';
