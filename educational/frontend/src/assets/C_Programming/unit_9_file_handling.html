Unit 9: File Handling in C

1. Introduction

File handling in C allows us to store data permanently in a file. Files are stored on the disk and can be read from or written to using file handling functions.

2. Types of Files

  • Text Files: Human-readable format
  • Binary Files: Non-human-readable format, used for faster access and smaller file size

3. File Operations

  • Creating a file
  • Opening a file
  • Reading and writing data
  • Closing a file

4. File Pointers

All file operations are performed using a FILE pointer.

FILE *fp;

5. Opening Files

fp = fopen("filename.txt", "mode");
ModeDescription
"r" Open for reading. File must exist.
"w" Open for writing. Creates new or overwrites existing file.
"a" Append to existing file.
"r+" Open for both reading and writing.

6. Closing a File

fclose(fp);

7. Reading and Writing Characters

fputc('A', fp);           // write character
char ch = fgetc(fp);     // read character

8. Reading and Writing Strings

fputs("Hello", fp);        // write string
fgets(str, 50, fp);       // read string

9. Reading and Writing with Format

fprintf(fp, "%d", num);     // write formatted output
fscanf(fp, "%d", &num);    // read formatted input

10. Checking for End-of-File

while (!feof(fp)) {
  // process
}
